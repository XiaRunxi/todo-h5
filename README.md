# ToDoList 待办事项应用（Html5 + JS + CSS）

一个轻量、可离线使用、面向「今日待办」场景的单页应用。

## 目录结构

- `index.html`：页面结构与任务详情弹窗
- `style/index.css`：样式与深浅色主题
- `script/index.js`：任务逻辑、状态管理、渲染与本地存储

## 已实现功能

### P0 / MVP
- **任务创建**：支持标题输入，回车快速添加，默认加入今天。
- **今日待办视图**：显示今天任务与逾期未完成任务。
- **完成打勾**：可一键完成，完成后从今日视图移除。
- **日期设置**：支持今天/明天/本周/无日期，且支持自然语言日期识别。

### P1
- **任务详情编辑**：可编辑标题、备注、日期、优先级、标签、子任务。
- **优先级标记**：高/中/低，列表左侧颜色区分。
- **任务延期**：支持明天/后天/下周一快捷延期。
- **全部任务视图**：支持今日、收件箱（无日期）、全部三种视图切换。

### 推荐交互
- **极速创建**：输入框默认聚焦；`Ctrl/Cmd + N` 聚焦输入框。
- **默认排序规则**：逾期置顶，再按优先级（高→中→低），同级按创建时间倒序。
- **无日期任务收纳**：默认不进入今日视图，进入收件箱视图。

## 非功能特性

- **离线支持**：使用 `localStorage` 持久化任务，断网可用。
- **主题切换**：支持浅色/深色模式切换，且记忆用户选择。
- **新用户引导**：首次打开自动创建 3 条示例任务，可手动删除/归档。

## 使用方式

1. 直接双击 `index.html` 或用任意静态服务器运行。
2. 在输入框输入任务，例如：
   - `明天买奶茶`
   - `周五前交报告`
3. 点击任务「详情」进入编辑弹窗。
4. 使用「明天/后天/下周一」快捷按钮推迟任务。
